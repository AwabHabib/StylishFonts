Stylish Fonts Website

A web application built with HTML, CSS, and JavaScript that allows users to explore and apply a variety of stylish fonts to their text. Users can also bold the text for emphasis and see the changes in real-time. The website features a simple, responsive design suitable for all devices.

Features:

Font Selection: Browse and choose from a range of stylish fonts.
Text Customization: Bold the text for added emphasis.
Real-Time Preview: Instantly see how the selected fonts and bold styles affect your text.
Responsive Design: The website is optimized for both desktop and mobile devices, providing a seamless experience.
Installation

Clone this repository:
git clone https://github.com/your-username/stylish-fonts-website.git


Open the project folder and open index.html in your browser or use the Live Server extension in VS Code.

Enjoy customizing your text with a variety of stylish fonts!

Usage:

Choose a font from the list.
Enter your text in the input field.
Toggle the bold option to emphasize the text.
View your customized text in real-time.
